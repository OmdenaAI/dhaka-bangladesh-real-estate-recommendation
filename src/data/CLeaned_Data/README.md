All the cleaned CSV should be stored here.

The mandatory fields for Residential data are
  1. Area
  2. Building_Type
  3. Commercial_Type
  4. Location
  5. num_bath_rooms
  6. num_bed_rooms
  7. price
  8. purpose (Rental/Sale)
  
The mandatory fields for Commercial data are

  1. Area
  2. Building_type
  3. Commercial_Type
  4. location
  5. price
  6. purpose (Rental/Sale)
